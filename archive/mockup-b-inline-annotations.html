All endpoints accept and return JSON. Authentication is handled by the surrounding infrastructure layer, not by the review API itself.
Documents are the top-level resource. Each document contains components, which are the reviewable units of text extracted during ingestion.

The review surface supports annotations, open questions, evidence attachment, highlighting, and AI-assisted revision suggestions with full task-run provenance.
Endpoints 8 components
POST /api/ingest/file accepts a JSON body with name, format, and content fields to create a new document from text content.
1 annotation
PATCH /api/components/:id updates the current text of a component and creates an audited revision record with the previous text preserved.
POST /api/components/:id/ai-suggestions triggers the configured AI provider to generate a proposed text revision for the specified component.
2 proposals
GET /api/documents/:id returns the full document detail including all components, versions, and the current review state with annotations, questions, evidence, and AI suggestions.
POST /api/documents/:id/save creates a versioned snapshot of the current document state including all component text and review metadata.
Error Handling 3 components
Validation errors return 400 with a JSON body containing an errors array. Each entry describes the field and the constraint that was violated.
Resource-not-found errors return 404. Workflow constraint violations return 409 with a message explaining the blocked transition.
Provider errors during AI suggestion generation return 502 with the upstream error message preserved in the response body for debugging.
Versioning 3 components
Each save operation increments the document version number and stores a complete snapshot of all component text at that point in time.
Component-level edits create revision records that track the previous text, revised text, edit source, and optional AI suggestion reference.
Autosave snapshots are created automatically on review mutations and stored independently of explicit save versions.
